Class Overview

Classes and specializations are the basis of our combat system. Classes serve as a collection of specializations that share progression and certain traits, while specializations are what you use to fight.

These specializations are each designed with different playstyles and mechanics in mind. Additionally, they each come with numerous skills to use, all of which interact with each other to create a combat flow that feels connected and satsifying.

Key Features

A handful of key features that every class and specialization has in common.

Specialization Mechanics

Specializations and their skills come with different mechanics that are used to faciliate their playstyle, including special cooldowns, skill resources and use conditions.

No Skill Limits

Each specialization comes packed with many skills that all interact with each other. Luckily, you don't have to choose which ones you use. You can use all of them!

Universal Mobility

There are no skills that use mobility such as leaps or disengages. Instead, every specialization has a easy-to-use, omni-directional dash that allows them to evade incoming attacks.
