Versatile Casting



Casting Skills: [Swap Key] + [Key Bind]

With a heavy focus on combat in PROJECT Spellforged, casting skills needs to be fluid, and needs a good amount of slots for you to assign them to.

PROJECT Spellforged allows you to bind usable skills in 8 skill slots, and cast them in rapid succession using a well-paced casting system. This system allows you to use your Swap Key (Usually keybound to F) to prime an skill cast, then press a respective keybind to activate the skill.

With this system casting all of your skills is 2 key inputs away. The available inputs you have are as followed:

Skill Slot Combinations

Skill Slot # Key Combination Skill Slot # Key Combination
1 Swap Key + Left Click 5 Sneak + Left Click
2 Swap Key + Right Click 6 Sneak + Right Click
3 Swap Key + Drop Key 7 Sneak + Drop Key
4 Swap Key + Swap Key 8 Sneak + Sneak
