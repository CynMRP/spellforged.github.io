Frequently Asked Questions

Find answers to common questions about PROJECT Spellforged

Have questions about our project? Browse through our FAQ below. If you can't find what you're looking for, join our Discord and ask us directly.

When will PROJECT Spellforged be released to the public?

No official timeframe is confirmed yet. PROJECT Spellforged isn't something we want to rush, and we want at least over a year of development time to plan, create, and polish our experience. None of us on the team want to release an unfinished project because we had a due date to abide by.

Will PROJECT Spellforged have features like Guilds and Housing?

As of right now, features like those are not planned for official release. We want to create an initial experience that contains all of our required features and prerequisites before we start adding additional features like Guilds. While we plan to add these features in future updates, we need to analyze our playerbase and server load first to determine our priorities.

What Minecraft Version will PROJECT Spellforged be hosted on?

PROJECT Spellforged will always be hosted on the latest version of Minecraft (currently 1.21.x Java Edition). We want to use the latest version to give ourselves all of the latest resources Minecraft has to offer to build our world, and to allow players to play without having to downgrade versions.

How can I keep updated on PROJECT Spellforged?

The best way to stay updated is to join our Discord! There you can meet and converse with the team, get the latest updates about the project, and even have the chance to apply to help build the project.

Will I get the chance to test out PROJECT Spellforged?

When we are nearing the end of closed development, we will have a Closed Alpha/Beta test. Testers will be handpicked to ensure quality feedback and manageable server loads. This helps us identify bugs and validate that our content and systems are fun and enjoyable.

Who is directing the project?

The project is led by Novalescent, who serves as the Developer, Game Designer, and Project Director. With extensive experience in Minecraft development and multiple successful projects, the team is committed to creating an exceptional MMORPG experience.
